# Flying-Plane
Jeu Unity 3D où vous avez le contrôle d'un avion qui doit éviter des murs.
Vous avez la possibilité de le faire monter ou descendre.

Ce jeu a été créé en suivant le site officiel https://learn.unity.com/pathway/junior-programmer/unit/player-control/tutorial/challenge-1-plane-programming-1?version=6.0

Vous pouvez tester le jeu avec ce lien : https://play.unity.com/en/games/c3200862-53eb-4572-a107-2d3b3b3b58be/exportwebavion
